# 📚 Study Buddy AI

Study Buddy AI is an intelligent homework assistant app.

Users take a picture of a full homework paper, and the AI:

- Extracts all questions from the image
- Separates each question clearly
- Provides correct answers
- Gives step-by-step explanations
- Supports multiple subjects (Math, Biology, English, Science, etc.)

---

## 🚀 How It Works

1. User takes a photo of homework.
2. The app extracts text using OCR (Optical Character Recognition).
3. The extracted text is sent to an AI model.
4. The AI:
   - Identifies all questions
   - Answers each one
   - Provides explanations
5. Results are displayed clearly inside the app.

---

## 🛠 Tech Stack

- React Native (Expo)
- OCR for text extraction
- AI language model for reasoning
- Firebase (planned for backend)

---

## 🎯 Goal

To help students understand their homework better by providing clear explanations — not just answers.
